# 🐧 lazywslc

A **lazydocker-inspired** TUI dashboard for managing WSL Linux containers via `wslc.exe`.

## Features

- Three-panel layout for Containers, Images, and Volumes
- Live CPU/memory sparkline graphs
- Combined logs and stats view
- Mouse and keyboard support
- Context actions (start/stop/kill/remove/prune)
- Auto-refresh

## Install

Grab the latest `.msi` or `.exe` from the [Releases](https://github.com/crloewen/lazywslc/releases) page.
